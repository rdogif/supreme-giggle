# Rishaan Personal Website

A simple personal portfolio website made with HTML, CSS, and JavaScript.

## How to use it on GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `style.css`, and `script.js`.
3. Open the repository settings.
4. Go to **Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the `main` branch and `/root`.
7. Save and wait for the GitHub Pages link to appear.

The main file must be named `index.html`.
